Using buttons

  • Use an icon on buttons, unless it's not possible or if the button is being used as a tag / category link
  • Use solid icons (fas) on solid buttons
  • Light icons (fal) on basic and outline buttons
  • Use solid buttons for key actions like saving, submitting, or deleting
  • Basic and outline buttons are for secondary actions
    • Basic buttons are preferred
    • Outline buttons are useful in groups and as a clickable tag or category
