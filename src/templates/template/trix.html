Trix editor

intergrate Trix editor in forms

To submit the contents of a --trix-editor-- with a form, first define a hidden input field in the form and assign it an id. Then reference that id in the editor’s input attribute.

Populating --trix-editor-- With Stored Content

To populate a trix-editor with stored content, include that content in the associated input element’s value attribute. NOTE: Always use an associated input element to safely populate an editor. Trix won’t load any HTML content inside a trix-editor & trix-editor tag.
